### About

I build production-grade AI and full-stack systems: from research notebooks and model training pipelines to deployed web and mobile products. My focus is shipping reliable software that turns data into measurable outcomes.

- Designing and deploying machine learning models for healthcare, finance, agriculture, and environmental monitoring.
- Building full-stack applications with React, Next.js, Node.js, and Flutter.
- Writing maintainable, well-tested code with a strong emphasis on performance, clarity, and developer experience.
- Comfortable across the stack: data engineering, model training, API design, frontend, deployment, and observability.

Currently exploring large language models, retrieval-augmented generation, MLOps, and scalable cloud-native architectures.
